D GREASER DETAILING CAR WASH - ANDROID STUDIO PROJECT

WHAT THIS PROJECT DOES
- Offline HTML app inside Android
- D Greaser logo
- Customer entry and exit
- Service pricing
- Earnings
- Date/month reports
- Individual receipt printing
- LocalStorage data

HOW TO BUILD APK
1. Install Android Studio.
2. Open this project folder in Android Studio.
3. Wait for Gradle Sync to finish.
4. Go to Build > Build Bundle(s) / APK(s) > Build APK(s).
5. The APK will be generated under:
   app/build/outputs/apk/debug/app-debug.apk

IMPORTANT
The HTML is stored in:
app/src/main/assets/index.html

The logo is stored in:
app/src/main/assets/logo.png
